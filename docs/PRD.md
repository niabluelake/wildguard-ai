# WildGuard AI PRD

## 1. 제품 개요

WildGuard AI는 야생동물 출몰 위험을 사전에 확인하고, 이미지/영상 탐지와 대응 상담을 통해 사용자에게 실질적인 행동 요령을 제공하는 AI 기반 위험 대응 지원 서비스이다.

---

## 2. 문제 정의

농장, 산림, 국립공원, 마을 주변에서는 야생동물 출몰로 인한 안전 문제와 시설물 피해 가능성이 존재한다. 그러나 일반 사용자는 현재 조건에서 위험도가 높은지, 어떤 행동을 해야 하는지 판단하기 어렵다.

WildGuard AI는 관측 데이터 기반 위험도 예측과 탐지, 상담 기능을 결합하여 사용자의 의사결정을 지원한다.

---

## 3. 목표

- AI Hub 야생동물 관측 데이터를 활용한 지역 기반 위험도 예측
- YOLO 기반 이미지/영상 야생동물 탐지
- 위험도 예측 결과 기반 대응 상담 제공
- 로그인 사용자의 예측 기록 저장 및 조회
- 향후 GPS, 신고 데이터, 실시간 기상 데이터와 연동 가능한 구조 설계

---

## 4. 핵심 사용자 시나리오

### 시나리오 1. 지역 위험도 확인

1. 사용자가 `/risk` 화면에 접속한다.
2. 관측 지역, 날씨, 시간대, 계절을 선택한다.
3. 시스템이 위험 점수와 위험 등급을 예측한다.
4. 사용자는 주요 위험 동물과 대응 요령을 확인한다.

### 시나리오 2. 이미지/영상 탐지

1. 사용자가 `/vision` 화면에 접속한다.
2. 이미지 또는 짧은 영상을 업로드한다.
3. 시스템이 YOLO 모델로 야생동물을 탐지한다.
4. 사용자는 bbox, confidence, 결과 이미지를 확인한다.

### 시나리오 3. 대응 상담

1. 사용자가 `/chat` 화면에 접속한다.
2. 지역 조건과 질문을 입력한다.
3. 시스템이 위험도 예측 결과를 바탕으로 상담 답변을 생성한다.
4. 사용자는 상세 상담 또는 현장 대응 안내를 확인한다.

---

## 5. 주요 기능

| 기능 | 사용자 가치 |
| --- | --- |
| 위험도 예측 | 특정 조건에서 야생동물 출몰 위험을 사전에 확인 |
| 이미지/영상 탐지 | 실제 촬영 자료에서 야생동물 존재 여부 확인 |
| 대응 상담 | 위험 상황에서 무엇을 해야 하는지 안내 |
| 예측 기록 | 과거 예측 결과를 저장하고 다시 확인 |

---

## 6. 성공 기준

| 항목 | 기준 |
| --- | --- |
| 위험도 예측 API | 정상 입력 시 위험 점수, 등급, 메시지 반환 |
| 웹 화면 | `/`, `/risk`, `/vision`, `/chat` 접근 가능 |
| 상담 API | 질문 입력 시 위험도 기반 답변 반환 |
| 모델 학습 | `area_risk_model.pkl` 생성 가능 |
| Git 관리 | 데이터와 모델 파일 제외, 코드와 문서만 관리 |

---

## 7. 현재 제약사항

- 현재 위험 점수는 실제 사고 데이터가 아니라 관측 데이터 기반 설계 점수이다.
- 현재 위치 기반 실시간 서비스가 아니라 AI Hub 관측 지역 기반 예측이다.
- 데이터 분포상 멧돼지 비중이 매우 높아 주요 위험 동물이 멧돼지로 편향될 수 있다.
- 모델과 원본 데이터는 GitHub에 업로드하지 않는다.

---

## 8. 향후 개선

- GPS 기반 현재 위치 예측
- 지역별 신고 데이터 연동
- 실시간 날씨 API 연동
- 위험 지도 시각화
- 이미지 탐지 결과와 위험도 예측 결과 통합
- 상담 답변 고도화
